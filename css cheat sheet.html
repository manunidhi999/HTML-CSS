⭐ CSS Complete Cheat Sheet (All Selectors + Units + Global Keywords)

1. Basic Selectors

* → Universal selector p → Element selector .box → Class selector #title → ID selector h1, h2 → Grouping

2. Combinators

div p → descendant div > p → direct child h1 + p → adjacent sibling h1 ~ p → general sibling

3. Pseudo-Classes

:hover, :active, :focus, :checked, :disabled, :visited, :first-child, :last-child, :nth-child(n), :not()

4. Pseudo-Elements

::before ::after ::first-letter ::first-line ::placeholder ::selection

5. Attribute Selectors

a[href] → has attribute a[href="google.com"] → exact match a[href^="https"] → starts with a[href$=".pdf"] → ends with a[href*="login"] → contains

6. CSS Property Categories

Layout

display, position, z-index, float, clear, overflow

Box Model

width, height, padding, margin, border, box-sizing

Flexbox

display:flex, justify-content, align-items, gap, flex-wrap

Grid

display:grid, grid-template-columns, grid-template-rows, gap

Typography

color, font-size, font-weight, text-align, text-decoration

Background

background-color, background-image, background-size

Effects

opacity, filter, transform, transition, animation

7. CSS Units

Length Units: px, %, em, rem, vh, vw, ch, ex, fr Color: hex, rgb(), rgba(), hsl() Time: s, ms Angle: deg, rad, turn

8. Global CSS Keywords

inherit – takes parent value
initial – reset to default value
unset – remove (initial + inherit)
auto – browser decides
none – no value
revert – revert to user-agent stylesheet
revert-layer – revert previous layer
